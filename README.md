# Portfolio Website

## This is my personal developer portfolio build with Next.js. It showcases my projects, technical skills, and experience as a full-stack software engineer.

## This website includes a contact form that sends emails directly to me using the Resend email API.

# Features

- Home page introducing my background and tech stack
- Projects page highlighting my development work
- Skills page outlining technologies I use
- Experience page covering my externship and bootcamp training
- Contact form with working email delivery

# Tech Stack

- Next.js
- React
- TypeScript / JavaScript
- Tailwind CSS
- Resend (email API)
